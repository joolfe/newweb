How to execute a SQL script in phpMyAdmin

Introduction

In many of the Opencart mod´s is necessary the modification of the Opencart DB Opencart for the proper functioning of the mod. Usually this modification comes in a file with the mod installation. We are going to explain how to execute this modification with the DB manager phpMyAdmin.

1.Log in into phpMyAdmin

phpMyAdmin is the DB that comes installed with the most hosting with mysql, we must to look for the icon in our "hosting control panel" like cpanel/GNUPane/VHCS/etc.., normally this is the Advance or DB box.

  Tip: Also is very easy press "ctrl + F" in the browser and look for "phpmy..." ;-)

2.Locating our Data Base

Once we are into phpMyAdmin we can see a screen like the picture (the view can be different depends of our hosting phpMyaAdmin version).

In left area we can see a list with our Dates Bases, we must select the Data Base with our opencart shop’s data

3.Execute the Script

Once we have access to the DB we can execute the script, the script (SQL code) must to be in the "Installation instructions" about our mod or in a ".sql" file, this files can be opened with a normal text editor, we’ll see something like:

  ALTER TABLE `product`  ADD COLUMN  `some_column` tinyint(1) COLLATE utf8_bin NOT NULL;

  ALTER TABLE `category`  ADD COLUMN  `another_column` int(11) COLLATE utf8_bin NOT NULL;

We copy the code and go to the SQL Tab, then we paste the script and we press the "Go" button that is bottom of screen (look the picture).

phpMyAdmin will show us a successful message! Please review the frequent problems paragraph for help.
