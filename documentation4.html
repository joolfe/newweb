How to configure a Layout in Opencart

What is a layout?

In Opencart a layout is a distribution of modules for one or more page (collection of pages) of your web. What does this mean? This mean than a layout is that allow you to have a module distribution for one page like this:

And a different one for other page likes this:

Defaults Layouts

In Opencart we have preconfigured layouts that in most cases be sufficient for a custom configuration, since it takes into account the different section of a store:

  • Account
  • Affiliate
  • Category
  • Checkout
  • Contact
  • Default
  • Home
  • Information
  • Manufacturer
  • Product
  • Sitemap

With opencart default layouts we can configure one mods in the home, other ones in the category page, even another one in the product pages etc... But sometimes we can want a greater level of settings for example in a new product we can differentiate from other products configuring a banner, for this case opencart let us create and configured new layouts..

Creating a new Layout

To administrate layouts we have to go to "System > Design > Layouts", we’ll see a list with the different predefined and created by us layouts, here we can create, edit and delete a layout like in any other administration page in opencart. If we press the "Insert" button we’ll see a form page like this:

Here we can configure the next fields:

Layout Name
Just a name to identify the layout
Store
The store where we want the layout.
Route
This is a default route to apply the layout, but you can apply specific layouts in products, categories... just configurations in product form. (See next step)

Configuring layouts for one opencart element (product, category, info...)

Layouts are applied by default to the page that’s match with the layout "Route" properties, but it is possible apply a layout to one or more specific opencart elements. Just in the form page of each element in the "Design" tab we can choose from the combo the layout we can for this element.

With this simple action we can change totally the aspect of a product, category...
